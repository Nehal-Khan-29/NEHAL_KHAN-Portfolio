AEyeD - Artificial Eye Decipher

A Computer Vision and IoT-based project that delivers an affordable, AI-powered real-time monitoring system to assist visually impaired individuals. AEyeD supports face and gesture recognition, vehicle and stair detection, and text-to-speech conversion, enabling safe navigation and enhanced situational awareness.

Developed using CNNs, Haar Cascade, OpenCV, Keras, and pytesseract, the system runs on a Raspberry Pi and includes voice-command-based face labeling and OCR-driven audio feedback. Tested with patients suffering from Retinitis Pigmentosa, AEyeD achieved a high F1 score of 0.88, outperforming non-assisted scenarios and showcasing its effectiveness in enhancing autonomy.

FACT - Fabrication Avoidance using Authentication and Encryption Techniques

FATE (Fabrication Avoidance using Authentication and Encryption Techniques) is a security-focused system that adopts a hybrid cryptographic approach to ensure both data integrity and authentication. It combines AES and RSA for robust encryption, while digital signatures with SHA-256 hashing are used to authenticate data. To future-proof the system, FATE integrates post-quantum algorithms like Kyber for secure key encapsulation and Dilithium for digital signatures, making it resilient against both classical and quantum attacks.

Upgrade

A Recommender system using ML-KNN algorithm to improve students learning experiences. The critical challenge is to identify the effective methods to enhance student learning outcomes. Some students learn from their mistakes, but the question is how they can be guided to do so efficiently. Without a proper guidance, students may choose less effective ways to correct their errors. Thus losing their chance to gain more marks. Hence there is a need to have a recommender system that can improve students learning experience.

Teaching staff provide a set of learning strategies tailored to different ranges of student marks, along with course details and student performance data. Using this information, the system generates a synthetic dataset that captures a diverse range of student mark distributions. A model is then trained on this dataset. Once trained, the model can recommend appropriate learning strategies to students based on actual test data, thereby guiding them more effectively and helping them improve their academic outcomes.
